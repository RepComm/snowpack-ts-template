
# snowpack-ts-template

My personal template for using:
- TypeScript
- ES modules
- npm packages
- d.ts files
- publishing to npm

All accomplished with [snowpack](https://www.snowpack.dev/)!

## scripts
### start
Local http dev server (uses `snowpack dev`)
`npm run start`
### build
`npm run build`
### build-types
`npm run build-types`
### publish
`npm run publish`
